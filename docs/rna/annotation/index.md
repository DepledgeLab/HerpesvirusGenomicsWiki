# transcriptome annotation

High-resolution transcriptome annotations are a pre-requisite for advanced transcriptomic analyses, especially where long-read sequencing approaches (PacBio, nanopore DRS) are used. Existing tools that are commonly used for genome annotation in higher eukaryotes (e.g. Bambu, IsoQuant, FLAIR, etc.) perform poorly against viral transcriptomes and are generally not fit for purpose. This led to the development of several tools that are optimised for herpesviral annotation projects. Tools such as LORTIA, NAGATA, and TRIMD have been developed to facilitate herpesvirus transcriptome annotation.

- [NAGATA (Nanopore Guided Annotation of Transcriptome Architectures)](https://journals.asm.org/doi/10.1128/msystems.00505-24){:target="_blank"} is a tool built specifically around nanopore direct RNA sequencing data, designed to tackle the challenge of annotating compact, gene-dense transcriptomes — viral genomes being the flagship use case, where overlapping ORFs and closely packed transcripts make conventional annotation approaches struggle. It works from the premise that each DRS read represents a single native RNA molecule and can therefore carry all the information needed to identify its genomic origin, along with features like modified bases and poly(A) tail length. It's tuned for rapid, high-resolution annotation of viral and other densely coded transcriptomes rather than being a general-purpose isoform caller.

- [LoRTIA (Long-read RNA-Seq Transcript Isoform Annotator)](https://www.nature.com/articles/s41598-021-93593-y){:target="_blank"} is a toolkit that takes SAM/BAM alignments from either Nanopore or PacBio long reads and outputs GFF-format transcript annotations, built around the idea that a transcript should only be trusted if it's supported by reads with correct, chemistry-appropriate adapter signatures at both ends. It identifies transcripts by searching for reads with correct 5′- and 3′-adapters that span from a high-confidence transcription start site to a high-confidence transcription end site, within a small nucleotide window — an approach that trades some sensitivity for high specificity, since reads lacking the right adapter signatures are excluded from transcript calling. It's been widely used for herpesvirus transcriptome work (e.g. HCMV, KSHV), and a newer "LoRTIA Plus" iteration extends this to a chemistry-agnostic, feature-first workflow across both cDNA and direct RNA datasets

- [TRIMD (Transcriptome Resolution through Integration of Multi-platform Data)](https://academic.oup.com/nar/article/44/18/e145/2468352?login=true){:target="_blank"} takes a different strategy: rather than relying on any single sequencing platform, it integrates data from three different transcriptomic platforms to resolve and validate full-length transcript isoforms, applying conservative calls that require strong corroborating evidence. It was developed for and applied to gammaherpesvirus transcriptomes (MHV68) and Epstein-Barr virus, where it substantially expanded the annotated EBV transcriptome by identifying hundreds of previously unrecognized transcripts. Because it demands multi-platform agreement, it tends to favor precision over discovery breadth compared to single-platform tools like NAGATA or LoRTIA.

Each of these tools is optimised for long-read sequencing inputs (nanopore, PacBio) and require alignment to a reference genomes (typically with minimap2 in splice-aware mode) prior to standard downstream processing of alignment files (e.g., [SAMTools](https://github.com/samtools/samtools){:target="_blank"}, [bedtools](https://bedtools.readthedocs.io/en/stable/){:target="_blank"}) to derive coverage profiles, splice junctions, and expression measures. 

The integration of multiple complementary approaches is often required to fully resolve herpesvirus transcriptomes. Short- and long-read RNA sequencing provide information on transcript abundance and structure. Other specialized methods provide precise mapping of transcription start and end sites (CAGE-Seq, circular/differential RNA-Seq) and distinguish coding from non-coding transcripts (e.g., ribosome profiling (Ribo-Seq)) while proteomic evidence (e.g., mass spectrometry, validated antibodies) provides direct support for protein expression. Interpretation of these datasets requires method-aware criteria. For example, ribosome profiling data should demonstrate triplet periodicity, initiation signatures, and response to chemical treatments (e.g., harringtonine), while transcript-based evidence should be supported by replicate data and robust splice-junction detection. Long-read sequencing is also particularly valuable for resolving transcript isoforms and 3’ ends since poly(A) tails are identified directly. Similarly, nanopore DRS-based approaches such as [Nanopore ReCappable Sequencing](https://academic.oup.com/nar/article/50/6/3475/6542485){:target="_blank"} and [TERA-Seq](https://pubmed.ncbi.nlm.nih.gov/34428294/){:target="_blank"} also offer the ability to directly capture transcription start sites. It should also be noted that specialized approaches are often required for [specific RNA classes](../nanopore-drs/alternative-approaches){:target="_blank"}. For example, small RNAs and highly structured transcripts (including tRNAs) may require alternative alignment strategies (e.g., BWA MEM). Similarly, analysis of RNA modifications requires dedicated tools (e.g., modkit and others) and careful filtering to distinguish biological signal from technical noise.
